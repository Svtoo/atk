# Security Policy

## Supported versions

Only the latest released version receives security fixes.

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Report privately via GitHub's [Security Advisories](https://github.com/Svtoo/atk/security/advisories/new)
or by emailing **sasha@svtoo.com**.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 7 days. Confirmed issues will be fixed promptly
and disclosed in a coordinated manner.
